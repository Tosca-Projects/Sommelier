### Running Sommelier
Prerequisites: 
 * `python` installed,
 * `pip` installed, and
 * `tosca-parser` installed (`sudo pip install 'tosca-parser'`).

To run Sommelier, just type `python sommelier.py --template-file=[topology-template-file-name]`
